Customer Clustering

Data acquired from filed forms and user surveys can be used to determine necessary insights on the segmented client groups. Using the K-means clustering algorithm, this data is processed to determine optimal clusters. The clusters are then dimensionally reduced and visualized with Principal Component Analysis (PCA).Custonova models and processes data into clusters. The clusters form a foundation from which the segments are defined.
